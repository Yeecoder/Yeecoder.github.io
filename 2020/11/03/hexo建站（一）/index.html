安装和初步使用Hexo

首先要先安装node.js和git（Windows下）

  • 安装node.js:
    进入node.js官网，下载之后一路next进行安装

  • 安装git:
    进入git官网，下载后同样一路next进行安装

安装完成后可以分别使用node -V和git来检验是否正常安装

之后输入npm install -g cnpm –registry=http://registry.npm.taobao.org 安装淘宝镜像，会极大的加快hexo的下载速度

完成后同样可以使用cnpm -V进行验证

使用npm install hexo-cli -g安装hexo

可以使用hexo -v查看hexo的版本

之后找一个合适的位置建立文件夹，作为博客的根目录。

使用hexo init blog建立并初始化博客站点

执行hexo s启动项目，在浏览器中打开localhost:4000即可看到hexo默认生成的页面

执行hexo n “文章名称”创建新文章

（在每次对文章进行修改或创建新的文章、页面后都需要执行hexo g

将博客部署至GitHub

Github创建一个新的仓库：XXXX.github.io（XXXX的部分要和自己的用户名相同）

执行cnpm install –save hexo-deployer-git 命令在blog目录下安装git部署插件

修改配置_config.yml

Deployment
  Docs: https://hexo.io/docs/deployment.html
  deploy:
    type: git
    repo: https://github.com/YourGithubName/YourGithubName.github.io.git
    branch: master

完成上述操作后，保存对配置文件的修改

执行hexo d将本地文件部署到Github仓库里

访问https://YourGithubName.github.io/可以查看博客

更改博客的主题

执行git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia 下载yilia主题到本地

修改hexo根目录下的 _config.yml 文件 ： theme: yilia

执行hexo g 重新生成即可查看修改后的主题和内容
