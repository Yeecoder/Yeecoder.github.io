题目

用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 )

示例1

输入：[“CQueue”,”appendTail”,”deleteHead”,”deleteHead”]
[[],[3],[],[]]
输出：[null,null,3,-1]

示例2

输入：[“CQueue”,”deleteHead”,”appendTail”,”appendTail”,”deleteHead”,”deleteHead”]
[[],[],[5],[2],[],[]]
输出：[null,-1,null,null,5,2]

限制

最多会对 appendTail、deleteHead 进行 10000 次调用
1 <= values <= 10000

我的题解

python

由于队列是先进先出的，而栈是先进后出的，则可以使用其中一个栈作为尾部输入，再将所有元素放入另一个栈进行逆置，从而达到先进先出的效果。

执行用时：440 ms, 在所有 Python3 提交中击败了75.07%的用户
内存消耗：18.8 MB, 在所有 Python3 提交中击败了5.06%的用户

C++

由于队列是先进先出的，而栈是先进后出的，则可以使用其中一个栈作为尾部输入，再将所有元素放入另一个栈进行逆置，从而达到先进先出的效果。

执行用时：304 ms, 在所有 C++ 提交中击败了99.74%的用户

优质解析

python

利用递归，先递推至链表末端；回溯时，依次将节点值加入列表，即可实现链表值的倒序输出。递归的终止条件： 当 head == None 时，代表越过了链表尾节点，则返回空列表；递推工作： 访问下一节点 head.next ；
